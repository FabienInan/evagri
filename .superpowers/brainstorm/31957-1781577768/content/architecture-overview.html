Points clés

  • Données personnelles et transactions hébergées au Québec, conforme Loi 25.
  • Déploiement via Coolify (interface web) pour le technicien de maintenance.

  • Une seule base de données relationnelle (PostgreSQL) pour toutes les entités métier.
